no-mouse search

Search google without using a mouse, just type your query and press enter.

how to use

  1. For Chrome install this plugin. (Because someone at Google, in all their wisdom, has decided that you lack the intelligence to determine what a new tab should look like.)
  2. Make this page as your new tab URL.
  3. Open a new tab with a keyboard shortcut. The default is ctrl+t, unless you're a mac user.
  4. Type in your query, the input is auto-selected.
  5. Press enter. (No, there is no submit button. This page is not intended for mouse users.)
